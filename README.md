# 火星企鵝：明亮探險版 (Mars Penguin: Bright Adventure)

這是一個使用 Three.js 構建的 3D 網頁遊戲。玩家扮演一隻在火星上探險的企鵝，目標是尋找並收集散落在地圖上的各種神祕物品。

## 遊戲特色

*   **3D 場景**：使用 Three.js 打造的火星地表，包含地形起伏、光影效果和霧氣氛圍。
*   **企鵝主角**：可愛的 3D 企鵝模型，帶有背包和探照燈。
*   **探索任務**：尋找 6 個特定的任務物品（前哨站、好奇號殘骸、能量水晶、石碑、化石、太陽能板）。
*   **雷達系統**：右上角的雷達可以幫助你定位附近的任務物品。
*   **計時挑戰**：記錄你完成探索所花費的時間。

## 如何遊玩

1.  **移動**：
    *   使用鍵盤 `W`, `A`, `S`, `D` 或方向鍵控制企鵝移動。
    *   點擊地面，企鵝會自動走到點擊的位置。
2.  **探索**：
    *   移動到物體附近（距離 < 6）會觸發發現事件。
    *   左上角的「探索日誌」會顯示已發現的物品。
    *   右上角的雷達會顯示附近的物品位置（不同顏色代表不同物品）。
3.  **勝利**：
    *   找齊所有 6 個任務物品即可通關。
    *   通關後可以選擇「重新挑戰」或「繼續閒晃」。

## 專案結構

*   `index.html`: 遊戲主頁面。
*   `style.css`: 樣式表。
*   `script.js`: 遊戲邏輯與 Three.js 程式碼。
*   `README.md`: 專案說明文件。

## 技術棧

*   HTML5
*   CSS3
*   JavaScript (ES6+)
*   [Three.js](https://threejs.org/) (r128)

## 安裝與執行

由於使用了 ES Modules 或 CDN 資源，建議使用本地伺服器運行以獲得最佳體驗（雖然本專案目前只使用了單一 script 檔案，直接打開 `index.html` 通常也能運作，但為了避免 CORS 問題，推薦使用 Live Server）。

1.  使用 VS Code 安裝 "Live Server" 擴充套件。
2.  右鍵點擊 `index.html`，選擇 "Open with Live Server"。
